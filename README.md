# hyper-chain

Hyper-chain is a BlockChain platform that is useful for ***Scalability*** and ***Data Management***. this platform is offered as a ***Direct Installation*** and as a ***Docker Base***.

# project directory structure

```
├── consensus
├── network
├── restAPI
└── utils
```
